# 字节火山方舟使用教程

> **火山方舟** 是字节跳动旗下火山引擎推出的一站式大模型服务平台（MaaS），集成豆包及多家主流大模型，提供训练、推理、评测、精调等全流程服务。

---

## 一、注册账号

### 步骤 1：打开官网

访问火山引擎官网，进入火山方舟页面。

<p align="center"><img src="/tutorial-assets/doc-05/img-001.png" width="600"></p>

### 步骤 2：注册并登录

点击右上角 **「立即注册」**，使用手机号注册并完成验证。

<p align="center"><img src="/tutorial-assets/doc-05/img-002.png" width="600"></p>

---

## 二、进入控制台

### 步骤 3：进入平台

点击「立即体验」，进入火山方舟控制台。

<p align="center"><img src="/tutorial-assets/doc-05/img-003.png" width="600"></p>

### 步骤 4：领取免费额度

点击左下角 **「领取 50w Tokens 免费」** 按钮。

<p align="center"><img src="/tutorial-assets/doc-05/img-004.png" width="600"></p>

---

## 三、获取 API Key

### 步骤 5：进入 API 接入页面

点击 **「API 接入」** 进入配置页面。

> **注意**：如果直接点击「立即体验」，可能会显示「当前对话 ID 不存在」，请改从 API 接入入口进入。

<p align="center"><img src="/tutorial-assets/doc-05/img-005.png" width="600"></p>

### 步骤 6：同意协议并获取 API Key

点击「同意」，即可获得 API Key。

<p align="center"><img src="/tutorial-assets/doc-05/img-006.png" width="600"></p>

### 步骤 7：实名认证 + 获取模型信息

1. 通过微信扫码完成 **人脸实名认证**
2. 实名后选择 **开通所有模型**
3. 在 **OpenAI SDK** 配置中找到 **URL** 和 **模型 ID**

<p align="center"><img src="/tutorial-assets/doc-05/img-007.png" width="600"></p>

<p align="center"><img src="/tutorial-assets/doc-05/img-008.png" width="600"></p>

---

## 四、创建应用

完成以上配置后，即可一键创建你想要的 AI 应用。
