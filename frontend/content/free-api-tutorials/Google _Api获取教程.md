# Google API 获取教程

> 本教程将指导你完成 Google 账号注册及 Google API 密钥的创建流程。

## 一、注册 Google 账号

> **前提条件**：电脑和手机挂好代理，保证节点一致且纯净；关闭一切杀毒软件、防火墙、电脑管家。

### 步骤 1：打开注册页面

访问 Google 账号注册页面，按提示填写信息。

### 步骤 3：完成手机验证

<p align="center"><img src="/tutorial-assets/doc-06/img-004.png" width="600"></p>

<p align="center"><img src="/tutorial-assets/doc-06/img-005.png" width="600"></p>

使用必应浏览器扫描二维码完成验证。

> **常见失败原因**：
> - 节点不够纯净（最常见）
> - 手机号已注册过
> - 手机号未开通国际通信服务（如校园卡）
