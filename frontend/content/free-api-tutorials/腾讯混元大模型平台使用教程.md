# 腾讯混元大模型平台使用教程

> 本教程将指导你完成腾讯混元大模型的注册、实名认证、API 密钥申请及接口 URL 获取的全流程。

---

## 一、平台核心网址

腾讯混元大模型官方开放平台：<https://hunyuan.tencent.com/>

### 步骤 1：进入登录界面

打开上述网址，点击页面 **右上角「控制台」**，跳转到腾讯云统一登录界面。

### 步骤 2：注册 / 登录腾讯云账号

- **已有账号**：选择「微信扫码」「QQ 扫码」或「账号密码」登录
- **无账号**：点击 **「立即注册」**，选择「微信/QQ 注册」或「手机号注册」，按提示完成验证

### 步骤 3：首次登录授权

首次登录混元控制台时，弹出 **服务授权提示**，点击「同意」即可进入控制台。

---

## 二、完成实名认证

> 调用 API 的核心凭证是 **SecretId** 和 **SecretKey**，申请前必须完成腾讯云实名认证。

1. 在控制台任意页面，若未认证会弹出提示，点击「去认证」
2. 也可手动访问：<https://console.cloud.tencent.com/developer/auth>
3. 认证类型：
   - **个人版**：上传身份证 + 人脸识别，约 5 分钟完成
   - **企业版**：提供营业执照、法人信息

---

## 三、申请 API 密钥（SecretId / SecretKey）

### 步骤 1：进入接入管理

登录控制台后，点击左侧菜单栏 **「接入管理」**（若菜单折叠，点击「☰」展开）。

<p align="center"><img src="/tutorial-assets/doc-04/img-001.png" width="600"></p>

### 步骤 2：了解密钥安全风险

系统会弹出主账号 API 密钥安全提示：

- **核心风险**：主账号密钥一旦泄露，他人可操控你账号下所有云资源
- **安全建议**：优先使用 **子用户 API 密钥** 替代主账号密钥（最小权限原则）
- 勾选「我已知晓使用主账号 API 访问密钥的风险」后继续

<p align="center"><img src="/tutorial-assets/doc-04/img-002.png" width="600"></p>

<p align="center"><img src="/tutorial-assets/doc-04/img-003.png" width="600"></p>

### 步骤 3：创建子用户

1. 选择 **「快速创建」**，输入用户名（如 `api_user_xxx`）
2. 勾选 **「编程访问」**（生成 API 密钥的必要选项）

<p align="center"><img src="/tutorial-assets/doc-04/img-004.png" width="600"></p>

### 步骤 4：配置子用户权限

在「权限设置」页面，点击「关联策略」，搜索并选择 `TencentCloudMLLMFullAccess` 等模型相关策略。

<p align="center"><img src="/tutorial-assets/doc-04/img-005.png" width="600"></p>

### 步骤 5：获取子用户 API 密钥

创建成功后，页面弹出包含 **SecretId** 和 **SecretKey** 的弹窗。

<p align="center"><img src="/tutorial-assets/doc-04/img-006.png" width="600"></p>

> **重要**：立刻复制并妥善保存，关闭弹窗后无法再次查看 SecretKey（只能重置）。

---

## 四、API 接口 URL（请求地址）

| 类型 | URL |
| --- | --- |
| **通用对话接口（推荐）** | `https://hunyuan.tencentcloudapi.com` |
| **广州地域（备用）** | `https://hunyuan.ap-guangzhou.tencentcloudapi.com` |

> 通用 URL 适配混元所有基础模型的对话、问答接口，开发时直接使用即可。其他地域暂未开放专属节点，不推荐使用。
