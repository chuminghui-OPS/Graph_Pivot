# 硅基流动 Token 免费领取教程

> 本教程将指导你在硅基流动平台完成注册、实名认证、领取免费额度，并获取 API 密钥。

---

## 一、核心准备

- 电脑 + 可收验证码的手机号 + 硅基流动账号
- 官网地址：<https://cloud.siliconflow.cn/>

---

## 二、注册 + 领取免费额度 + 实名认证

### 步骤 1：注册登录

打开官网，点击「登录」，输入手机号 → 获取验证码，完成注册并登录。

<p align="center"><img src="/tutorial-assets/doc-02/img-001.png" width="600"></p>

### 步骤 2：获取基础额度

新用户登录后 **自动获得 20 元免费基础额度**，无需手动领取。

### 步骤 3：实名认证

左侧菜单栏点击 **「个人中心」→「实名认证」**，用支付宝扫描弹出的二维码完成认证。

<p align="center"><img src="/tutorial-assets/doc-02/img-002.png" width="600"></p>

### 步骤 4：领取认证代金券

审核通过后，点击左侧 **「活动中心」→「认证专享礼」→「立即领取」**，获得 **16 元通用代金券**（自动到账，有效期 6 个月）。

<p align="center"><img src="/tutorial-assets/doc-02/img-003.png" width="600"></p>

### 步骤 5：查看资产

左侧 **「账户管理」** 可查看免费额度和代金券余额。

---

## 三、获取 API 密钥 + 模型 URL

### 步骤 1：创建 API 密钥

左侧菜单栏点击 **「API 密钥」→ 右上角「新建 API 密钥」**，输入描述后点击「新建」。

<p align="center"><img src="/tutorial-assets/doc-02/img-004.png" width="600"></p>

> **重要**：密钥仅显示一次，请立即复制并保存，丢失需重新生成。

### 步骤 2：搜索模型

左侧点击 **「模型市场」**，在搜索框输入 `hunyuan`，点击搜索。

<p align="center"><img src="/tutorial-assets/doc-02/img-005.png" width="600"></p>

### 步骤 3：获取模型信息

1. 选择对应模型（新手推荐 **混元 Lite**）
2. 进入模型详情页，切换至 **「API 调用」** 标签
3. 复制 **模型 URL** 和 **模型名称**

<p align="center"><img src="/tutorial-assets/doc-02/img-006.png" width="600"></p>

<p align="center"><img src="/tutorial-assets/doc-02/img-007.png" width="600"></p>

<p align="center"><img src="/tutorial-assets/doc-02/img-008.png" width="600"></p>

### 步骤 4：配置使用

将以下三项信息粘贴到调用工具的对应输入框：

| 配置项 | 说明 |
| --- | --- |
| **API 密钥** | 步骤 1 中创建的密钥 |
| **模型 URL** | 步骤 3 中复制的 URL |
| **模型名称** | 步骤 3 中复制的名称 |

---

## 四、16 元代金券使用说明

- 全平台通用，可抵扣所有模型调用费用
- 系统自动优先抵扣，无需手动操作
- 抵扣顺序：代金券 → 赠送余额 → 充值余额
- 查看记录：左侧 **「账单」→「消费明细」**
