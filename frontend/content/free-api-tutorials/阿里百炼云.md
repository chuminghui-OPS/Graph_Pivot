# 阿里百炼云使用教程

> **阿里百炼云（Model Studio）** 是阿里云推出的一站式企业级大模型服务平台，集成通义千问系列及主流第三方大模型，提供模型调用、微调、知识库构建等全流程工具链。

---

## 一、注册并开通服务

### 步骤 1：进入平台

访问阿里云百炼平台，点击 **「免费体验」** 进入。

<p align="center"><img src="/tutorial-assets/doc-01/img-001.png" width="600"></p>

### 步骤 2：注册账号

点击右上角 **「前往注册」**，使用手机号注册并设置密码。

<p align="center"><img src="/tutorial-assets/doc-01/img-002.png" width="600"></p>

### 步骤 3：完成开通

根据页面指示完成操作，刷新页面即可成功开通。

<p align="center"><img src="/tutorial-assets/doc-01/img-003.png" width="600"></p>

---

## 二、创建 API Key

> **注意**：开通百炼后需手动创建 API Key，系统不会自动生成。如弹出服务协议，阅读并同意后将自动开通。如提示未实名认证，请先完成实名认证。

### 步骤 4：生成 API Key

在控制台中找到 API Key 管理页面，点击创建。

<p align="center"><img src="/tutorial-assets/doc-01/img-004.png" width="600"></p>

---

## 三、配置使用

在第三方工具（如 Chatbox、Dify、Cline、Claude Code 等）中调用模型时，需要填写以下信息：

| 配置项 | 值 |
| --- | --- |
| **API Key** | 上一步创建的密钥 |
| **Base URL** | `https://dashscope.aliyuncs.com/compatible-mode/v1` |
| **模型名称** | 如 `qwen-plus` |
