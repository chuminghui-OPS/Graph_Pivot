# OpenRouter API 获取教程

> 本教程将指导你完成 OpenRouter 平台的注册及 API Key 获取流程。

---

## 步骤 1：访问官网并注册

访问 [OpenRouter 官网](https://openrouter.ai)，使用 **GitHub 账号**或 **Google 账号**登录注册。

<p align="center"><img src="/tutorial-assets/doc-07/img-001.png" width="600"></p>

## 步骤 2：进入 API Keys 页面

登录后，在控制台找到 API Keys 管理页面。

<p align="center"><img src="/tutorial-assets/doc-07/img-002.png" width="600"></p>

## 步骤 3：创建 API Key

点击创建新的 API Key，输入描述名称后确认。

<p align="center"><img src="/tutorial-assets/doc-07/img-003.png" width="600"></p>

## 步骤 4：复制并保存

<p align="center"><img src="/tutorial-assets/doc-07/img-004.png" width="600"></p>

> **重要**：API Key 仅显示一次，请立即复制并妥善保存。
